# Validation Checklist

생성 후 아래 항목을 확인한다.

- [ ] 초안에 약속한 파일이 모두 생성되었다
- [ ] 선택하지 않은 플랫폼 파일은 생성되지 않았다
- [ ] `CLAUDE.md`, `AGENTS.md`, `prompt.md`의 금지 행동이 일치한다
- [ ] `.claude/settings.json` 과 `skill.json` 이 유효한 JSON이다
- [ ] 생성된 `SKILL.md` 는 frontmatter와 필수 섹션을 가진다
- [ ] 스크립트는 shebang 과 `Usage:` 주석을 가진다
- [ ] `<...>`, `TODO`, 의미 없는 `...` 가 남아 있지 않다
- [ ] 시크릿이 파일에 직접 들어가지 않았다
- [ ] 승인 경계가 문서마다 흐려지지 않았다
- [ ] 리뷰어 핸드오프용 요약 또는 템플릿이 존재한다
