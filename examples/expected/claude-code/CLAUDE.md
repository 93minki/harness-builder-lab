# Project Harness

## Context

이 프로젝트에서는 버그 triage, 리뷰어 핸드오프, 스펙 초안 작성을 반복한다.

## Common Commands

- `rg`
- `bundle exec rspec`
- `pnpm test`
- `pnpm lint`

## Approval Policy

- 조사와 초안 작성은 자율 수행 가능
- 코드 수정, 브랜치 조작, 외부 네트워크 호출은 승인 필요

## Forbidden Actions

- `main` 직접 push
- 운영 데이터 변경
- 외부 서비스 메시지 전송

- `incident-triage`
- `reviewer-handoff`
